How to Run:

            1. Download the Code: They need to pull your code onto their local machine first.
                git clone https://github.com/connie4-beep/stock_dashboard.git
                cd stock_dashboard
            2. Create a Virtual Environment: They shouldn't install your packages directly into their system's main Python setup. They need to create an isolated environment so your dependencies don't conflict with their other projects.
                python -m venv venv

            3. Activate the Environment: They must "turn on" the environment.
                Mac/Linux: source venv/bin/activate
                Windows: venv\Scripts\activate

            4. Install the Requirements: Now they can safely run your installation command inside their isolated environment.
                pip install -r requirements.txt

            5. Set up the Environment Variables: Because you smartly excluded your .env file using .gitignore, it won't be in the repository they download. You need to instruct them to create their own .env file in the main folder and add their own GEMINI_API_KEY="your_api_key_here".

            6. Run the App: Finally, they can launch the dashboard.
                python app.py

Project: Implemented a hybrid image pipeline

Implemented a hybrid image pipeline for CS445: Computational Photography to explore image frequency representations and enhancement techniques. Created hybrid images by combining the high-frequency component of one image with the low-frequency component of another, producing images that change perception based on viewing distance. Applied frequency analysis using Fourier transforms to visualize input, filtered, and hybrid images. Extended the project with image enhancement tasks including contrast adjustment, color enhancement, and color shifts, using Python and OpenCV to manipulate images in HSV and LAB color spaces, improving both visual clarity and aesthetic quality.

  • Implemented hybrid image creation using Python by blending high- and low-frequency components, producing images that change perception with viewing distance
  • Applied Fourier transforms and frequency filtering (Gaussian and Laplacian) to analyze and visualize image components
  • Enhanced images through contrast, color, and hue adjustments using OpenCV, demonstrating control over color spaces and image enhancement techniques
